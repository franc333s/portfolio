
Documentation for the Portfolio

## MAP
To create classes for changing the text and background color, a map has been created in utilities.scss and variables in sass-variables.scss.

Example of classes:
Text: .text-color-black
Background: .bg-color-black



## FOR - revisar que no esta ja
To create classes that customize the width of a column, a @for loop has been created in utilities.scss and a variable in sass-variables.scss.

Example of class:
.column-width--1of8

    // Columns width
$parts: 8;
